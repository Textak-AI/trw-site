## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Deploy to Vercel

```bash
# Via GitHub (recommended)
git init && git add . && git commit -m "Initial site rebuild"
git remote add origin https://github.com/Textak-AI/trw-site.git
git push -u origin main

# Then import at vercel.com → New Project → Select repo
```
